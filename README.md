# Setup helm azure-tpl
Github action for setting up [helm azure-tpl plugin](https://github.com/webdevops/helm-azure-tpl) (requires Helm v3 or v4)

## Usage:

Latest version:
```
- uses: webdevops/setup-helm-azure-tpl@v1
```

Specific version
```
- uses: webdevops/setup-helm-azure-tpl@v1
  with:
    version: 0.63.7
```
